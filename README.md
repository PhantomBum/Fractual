# Fractual

Fractual is a readable, responsive ImGui-inspired placeholder menu. It fills its native desktop window and has no product purpose or application features.

## Install on Windows

Download and run `Fractual-Setup-x.y.z.exe` from the latest GitHub release. This initial installer is the last update that needs to be downloaded manually.

After installation, Fractual checks GitHub Releases automatically, downloads updates in the background, shows progress under **Settings → Auto update**, and offers **Restart** when the next version is ready. Downloaded updates also install when the app exits.

The menu uses readable surfaces, Inter Variable typography, a connected animated network background, responsive panel reflow, proper interface scaling, and an animated collapsible sidebar. UI Swap reproduces the Stellar opening interaction with a tactile cover-flow, enlarged center selection, dimmed adjacent previews, animated titles, arrow navigation, keyboard controls, and an apply action for the Classic, Stellar, Horizon, and Dock layouts.

Version 2.6 introduces two complete interface generations. **New** uses the redesigned floating layout, interactive connected background, new controls, and reworked layout variants. **Legacy** keeps the familiar 2.5 presentation. Both modes are available under **Settings → Interface mode**. Version 2.6 also removes the old panel focus rail that could appear as an unwanted group of lines near hovered rows.

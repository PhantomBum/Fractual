# Fractual

Fractual is a readable, responsive ImGui-inspired placeholder menu. It fills its native desktop window and has no product purpose or application features.

## Install on Windows

Download and run `Fractual-Setup-x.y.z.exe` from the latest GitHub release. This initial installer is the last update that needs to be downloaded manually.

After installation, Fractual checks GitHub Releases automatically, downloads updates in the background, shows progress under **Settings → Auto update**, and offers **Restart** when the next version is ready. Downloaded updates also install when the app exits.

The menu uses high-contrast surfaces, Inter Variable typography, a connected animated line background, responsive panel reflow, proper interface scaling, and an animated collapsible sidebar. UI Swap reproduces the Stellar opening interaction with a three-card cover-flow, enlarged center selection, dimmed adjacent previews, animated titles, arrow navigation, keyboard controls, and an apply action for the Classic, Stellar, Horizon, and Dock frames. Version 2.5 adds a secure GitHub Releases update channel, background download progress, restart-to-install, and an automated Windows release workflow while keeping the Showcase-inspired motion and connected background from v2.4.
